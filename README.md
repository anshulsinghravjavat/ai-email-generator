# 📧 AI Email Reply Generator

A full-stack AI-powered application that generates intelligent and context-aware email replies based on user input. The project supports multiple tones (professional, casual, friendly) and is designed to improve productivity in email communication.

---

## 🚀 Features

- ✨ Generate AI-based email replies
- 🎯 Multiple tone options (Professional, Casual, Friendly)
- ⚡ Fast and responsive UI
- 🔗 REST API integration
- 🧩 Chrome Extension support
- 🔐 Secure API key handling using environment variables

---

## 🛠️ Tech Stack

### Backend
- Spring Boot
- REST API
- Gemini API Integration

### Frontend
- React (Vite)
- Material UI
- Axios

### Extension
- Chrome Extension (JavaScript)
